Password Gen

Проект Берлина Андрея.

О проекте:

  • Я программист-любитель, программирую на JS и C#. Я разработал приложение "Password Gen" для создания пароля и его лёгкого запоминания.

  • "Password Gen" - поможет вам создать пароль и легко его запомнить по ключевому слову.

  • "Password Gen" - будет выходить в двух видах: 1) для браузера по ссылке и 2) для ОС Андроид в приложении APK.

  • В Твиттере проекта, будут публиковаться ссылки на приложения, и для дополнительной информации со мной можно связаться через Твиттер. Желаю удачи!
